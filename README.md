# 应用心理硕士 · 备考学习系统 🌸

西南交通大学 045400 应用心理（全日制/非全日制）备考学习系统。三门课：101 思想政治理论、204 英语（二）、347 心理学专业综合。

## 功能

- **📋 复习计划**：三阶段 25 项任务，覆盖政治、英语、347 心理学，勾选记录进度
- **🧠 思维导图**：三科知识结构可视化，章节可展开/收起、缩放、拖动平移，点知识点看详情
- **🎴 记忆卡片**：翻转式记忆卡，可按科目筛选、只看未掌握，"忘记/模糊/认识"三档自测
- **🌱 学习回顾**：总体与分阶段进度、知识掌握度、每日学习状态与时长、每周统计
- **📖 备考资料**：考试科目、347 参考书目、知识点关键词速查、招生官网链接

## 347 知识内容

内置约 60 个核心知识点，涵盖：

- **心理学导论/普通心理学**（黄希庭）：心理学流派、感知觉、记忆、思维、情绪、动机、能力、人格、学习
- **发展心理学**（林崇德）：皮亚杰、维果茨基、依恋、埃里克森、道德发展、自我同一性
- **心理与教育研究方法**（董奇）：实验设计、信度效度、抽样、心理测量、统计分析、研究伦理

## 使用方法

直接访问：https://christinechina33-pixel.github.io/psych-study-planner/

所有数据保存在浏览器本地存储，关闭页面后不丢失。支持"添加到主屏幕"作为 PWA 离线使用。

## 开发

`index.html` 由 `study-planner/build.py` 单一生成（唯一事实来源）。修改知识内容或界面后运行：

```bash
python study-planner/build.py
```
