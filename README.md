# 应用心理硕士 · 备考学习管理系统

西南交通大学 045400 应用心理（全日制/非全日制）备考学习计划管理工具。

## 功能

- **📋 复习计划**：三阶段25项任务，涵盖政治、英语、347心理学专业综合
- **📊 进度追踪**：总体进度 + 各阶段进度可视化
- **📝 学习回顾**：每日各科学习状态、学习时长记录、每周统计
- **ℹ️ 考试信息**：考试科目、参考书目、招生简章链接

## 使用方法

直接访问：https://christinechina33-pixel.github.io/psych-study-planner/
